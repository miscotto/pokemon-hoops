How should the new routing be structured?

Pick the URL structure that feels most natural for Pokemon Hoops

A

Clean Nested Routes

/tournaments — all tournaments list
/tournaments/[id] — specific tournament bracket
/rosters/[id]/build — roster builder
/profile — your own profile (wins, losses, history)
/users/[id] — view any player's profile

B

Dashboard-Anchored Routes

/dashboard — home hub
/dashboard/tournaments/[id] — tournament page
/dashboard/rosters/[id]/build — roster builder
/dashboard/profile — your profile
/dashboard/players/[id] — other player profiles

C

Flat Routes (Simple)

/tournaments/[id] — tournament bracket
/build/[rosterId] — roster builder
/player/[userId] — any user's profile (including yours)
No separate tournaments list page — use dashboard
